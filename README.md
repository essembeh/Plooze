```
usage: plooze
 -C,--cron <arg>       Run every X hours
 -c,--channel <arg>    Only search in a specific channel. Values: france2, france3, france4,
                       france5, franceo
 -d,--download <arg>   Download episodes to output folder
 -f,--fields <arg>     Match motif on given fields (defaults are
                       "id_diffusion,titre,accroche_programme")
 -F,--list-fields      List all available fields, see -f
 -h,--help             Display help
 -j,--json             Display json content while searching
 -o,--overwrite        Overwrite files if they already exist
 -q,--quality <arg>    Change the quality of downloaded files. Values: best, medium, lowest
 -v,--verbose          Display more information
 ```
